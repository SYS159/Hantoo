# 🇰🇷 Hantoo — 한국투자증권 자동매매 봇

> 라즈베리파이 위에서 돌아가는 한국 주식 자동매매 & 자산 모니터링 시스템

---

## 📦 설치

```bash
git clone https://github.com/SYS159/Hantoo.git
cd Hantoo
pip install requests python-dotenv
```

`.env` 파일 생성:

```env
APP_KEY=your_app_key
APP_SECRET=your_app_secret
ACCOUNT=your_account_number
DISCORD_WEBHOOK=your_webhook_url
```

---

## 🚀 실행

> **HM 먼저 실행** — 토큰 발급을 HM이 담당합니다

```bash
nohup python3 -u HM_v1_2.py > /dev/null 2>&1 &
nohup python3 -u HTD_v1_2.py > /dev/null 2>&1 &
```

## 📌 코드 구성

### 🔵 HM_v1_1 — 자산 모니터링
| 기능 | 설명 |
|------|------|
| 📊 자산 확인 | 장 중 매 시간 정각마다 계좌 현황 디스코드 전송 |
| 🔔 장 알림 | 개장(09:00) / 마감(15:30) 알림 |
| 🔑 토큰 관리 | 매일 자정 기준 토큰 자동 갱신 |

### 🟣 HM_v1_2 — 자산 모니터링 + 주간 리포트
HM_v1_1의 모든 기능 포함

| 기능 | 설명 |
|------|------|
| 📋 주간 리포트 | 매주 첫 장 날 09:00에 `trades.csv` 읽어서 주간 수익 보고 |
| 🏖️ 휴장 대응 | 월요일 휴장 시 화/수/목... 첫 장 날 자동 전송 |

---

### 🟡 HTD_v1_1 — 급등주 자동매매 (기본)
| 조건 | 값 |
|------|-----|
| 진입 | 등락률 +5% 이상 |
| 익절 | +3% 도달 후 트레일링 → 최고가 대비 -1% |
| 손절 | -2% |

### 🟢 HTD_v1_2 — 급등주 자동매매 (고도화)

**📈 매수 조건 (09:05 ~ 10:30)**
| 조건 | 값 |
|------|-----|
| 등락률 | +5% 이상 |
| 체결강도 | 120 이상 |
| 거래량 (09:05~09:30) | 전일 대비 **2배** 이상 |
| 거래량 (09:30~10:30) | 전일 대비 **3배** 이상 |
| 예수금 | 10만원 이상 시 매수 |

**📉 매도 조건**
| 조건 | 동작 |
|------|------|
| 진입 후 -2% | 🔴 손절 |
| 진입 후 +3% 도달 | 트레일링 스탑 활성화 |
| 최고가 대비 -1% | 🟡 익절 청산 |
| 15:20 | 🏁 보유 종목 전체 강제 청산 |

**📁 CSV 기록**
```
날짜 | 종목명 | 종목코드 | 매수가 | 매도가 | 수량 | 수익금(원) | 수익률(%) | 매도사유
```

## 💻 자주 쓰는 명령어

### Git

```bash
# 올리기
git add 파일명.py
git commit -m "메시지"
git push

# 내리기
git pull
```

### 라즈베리파이

```bash
# 실행 중인 봇 확인
ps -ef | grep python

# HM 재시작
pkill -f HM_v1_2.py
nohup python3 -u HM_v1_2.py > /dev/null 2>&1 &

# HTD 재시작
pkill -f HTD_v1_2.py
nohup python3 -u HTD_v1_2.py > /dev/null 2>&1 &
```

> `-u` 플래그가 있어야 로그가 실시간으로 기록됩니다
